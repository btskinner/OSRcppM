# OSRcppM


Install the latest development version from Github with

```r
devtools::install_github('btskinner/OSRcppM')
```

**NB:** Must have working installation of [OSRM backend](https://github.com/Project-OSRM/osrm-backend) on machine. At the moment, this repo assumes installation on MacOS using homebrew:

```bash
brew install osrm
```
